# 🐟 MiroFish Guide — OpenClaw Skill

Practical usage guide for [MiroFish](https://github.com/666ghj/MiroFish) swarm intelligence prediction engine.

## What This Skill Covers

- **Seed Text Design** — how to write effective seed texts for entity extraction
- **Agent Roles** — understanding auto-generated personas (MBTI, stance, influence)
- **Simulation Tuning** — rounds, activity levels, OOM prevention
- **Report Analysis** — working with ReportAgent and Zep tools
- **Codex Proxy Integration** — free GPT-5.4 access via ChatGPT Plus subscription
- **Lessons Learned** — real experiment logs (v1-v4) with bugs found and fixed

## Installation

### OpenClaw
```bash
openclaw skill install mirofish-guide.skill
```

### Manual
Copy `SKILL.md` and `references/` to your skills directory.
